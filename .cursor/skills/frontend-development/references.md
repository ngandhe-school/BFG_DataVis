## Distilled additions in this Cursor skill

- Define UI direction before coding (purpose, tone, constraints, differentiation).
- Add React anti-pattern guardrails (derived state, callbacks over setters, lean `useEffect` use).
- Add accessibility baseline checks (semantic structure, keyboard, focus, contrast).
- Keep browser-level verification as a required completion step.
